What to check:
  1. Hover over "TEST MENU" - dropdown should appear below
  2. Hover over "Submenu Item" (has a star icon) - nested menu should appear to the RIGHT
  3. The nested menu should be positioned next to the "Submenu Item" text
If it's NOT working:
  • Press F12 to open DevTools
  • Go to Elements tab
  • Hover to open the nested menu
  • Inspect the nested <ul class="dropdown-menu">
  • Check the "Styles" panel for any style="..." attributes
  • Look for positioning values like left, top, transform
