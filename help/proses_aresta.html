PROSES TUTUP ARESTA

Modul : Kebun
Tentang : Proses Tutup Aresta
Isi :

Langkah-langkah untuk proses tutup aresta :

  1. Masuk ke menu, pilih kebun dan tahun kemudian klik tombol Simpan untuk menampilkan list data areal statement

    2. Sesuai kebutuhan Medco, proses dilakukan per bulan, bukan per tahun pilihan tahun diganti menjadi periode.

  2. Berikut List Data Areal Statement, perhatikan catatan pada List data :

    3. Klik pada kolom mutasi untuk menampilkan detail mutasi. Kemudian klik tombol Proses untuk melakukan proses tutup areal statement.

  3. Jika saat klik tombol proses terdapat keterangan dibawah ini maka sebelumnya sudah pernah melakukan proses tutup Aresta di Periode yang sama :
