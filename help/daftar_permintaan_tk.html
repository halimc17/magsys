DAFTAR PERMINTAAN TENAGA KERJA

Modul : SDM
Tentang : Daftar Permintaan Tenaga Kerja
Isi :

Langkah-langkah Persetujuan Permintaan Tenaga Kerja oleh HRD :

  1. Masuk ke menu, akan muncul Daftar Permintaan Tenaga Kerja seperti di bawah ini :

  2. Isi tanggal sebagai tanggal akhir display lowongan pada website karir.minanga.co.id kemudian klik tombol Confirm untuk melakukan konfirmasi.

  3. Berikut tampilan permintaan tenaga kerja yang sudah dikonfirmasi :

    4. Jika tanggal akhir display lowongan ingin diperbarui maka isikan tanggal akhir kemudian klik tombol Memperbarui.

    Terdapat filter data permintaan tenaga kerja berdasarkan tahun.

  4. Klik icon untuk cetak Permintaan Tenaga Kerja dalam format PDF.

Langkah-langkah Persetujuan Permintaan Tenaga Kerja oleh Dept. Head selain HRD :

  1. Masuk ke menu, akan muncul Daftar Permintaan Tenaga Kerja seperti di bawah ini :

  2. Isi tanggal sebagai tanggal akhir display lowongan pada website karir.minanga.co.id kemudian klik tombol Confirm untuk melakukan konfirmasi.

  3. Berikut tampilan permintaan tenaga kerja yang sudah dikonfirmasi :

    4. Terdapat filter data permintaan tenaga kerja berdasarkan tahun.

  4. Klik icon untuk cetak Permintaan Tenaga Kerja dalam format PDF.
