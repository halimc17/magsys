SURAT JALAN

Modul : Pengadaan
Tentang : Pengiriman Internal/Surat Jalan
Isi :

Langkah-langkah input Surat Jalan :

  1. Masuk ke menu, akan terdapat form inputan seperti gambar dibawah ini klik buat baru untuk menginput transaksi Surat Jalan :

  2. Lengkapi data yang dibutuhkan nomor surat jalan akan terbentuk secara otomatis dan kode organisasi di disable berdasarkan lokasi tugas pembuat transaksi surat jalan tersebut, dan untuk field Diterima Oleh akan mengunci pada nama User yang memposting saat sudah selesai menyimpan data Surat Jalan di kolom List data

  3. Untuk mendapatkan detail dapat mengambil data dari PO, Package List, dan Material List dengan klik tombol masing-masing data detail tersebut

  4. Berikut tampilan detail untuk data berdasarkan No. PO, klik tombol Add Detail From PO akan muncul form masukkan No.PO atau tahun klik No.PO yang dimaksud setelah itu Add detail maka data akan masuk secara otomatis ke dalam tabel detail :

  5. Lakukan hal yang sama jika ingin menambahkan detail untuk Package List dan Material List dengan klik tombol detail yang ingin ditambahkan, input No. dokumen lakukan pencarian lalu add detail.
  6. Setelah add detail selanjutnya pada List Detail isi kolom jumlah, maka tampilan kolomnya secara otomatis akan menjadi seperti gambar dibawah ini, klik tombol simpan apabila sudah selesai input jumlahÂ 

  7. Klik List Data untuk melihat List Transaksi Surat Jalan yang sebelumnya sudah diinput, selanjutnya lakukan posting terlebih dahulu lalu klik tombol posting bulat berwarna merah pastikan sebelum klik tombol posting dokumen lengkap
  8. Sebagai catatan untuk kolom tanggal tiba akan terisi ketika sudah melakukan posting di list data surat jalan dan posting di Penerimaan Surat Jalan :
  9. Berikut tampilan data sebelum posting di surat jalan

  10. Berikut tampilan data setelah posting di surat jalan nama release by akan terisi oleh user penginput, sedangkan tanggal tiba terisi setelah posting di menu Penerimaan Surat Jalan

  11. Dapat melakukan pencarian data dengan menginput Nomor surat jalan yang diinginkan pada form pencarian lalu klik tombol cari seperti gambar dibawah ini :
