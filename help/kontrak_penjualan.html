Modul : Pemasaran
Tentang :Transaksi Kontrak Penjualan
Isi :

Langkah-langkah input Transaksi Kontrak Penjualan :

  1. Masuk ke menu akan terdapat form pengisian seperti dibawah ini :

  2. Lengkapi form kemudian klik tombol simpan untuk menyimpan data atau batal untuk membatalkan transaksi :

  3. Data yang sudah berhasil tersimpan akan muncul pada tab List data berikut ini :

  4. Terdapat aplikasi untuk mencari data berdasarkan No.Kontrak dan pilihan Perusahaan

  5. Klik icon untuk mengedit data Transaksi Kontrak Penjualan
  6. Klik icon untuk menghapus data Transaksi Kontrak Penjualan
  7. Klick icon untuk melihat data Transaksi Kontrak Penjualan dalam format PDF
  8. Data Transaksi Pemasaran Kontrak Penjualan ini juga dapat dilihat dalam format Excel dengan klik tombol excel pada kolom action
