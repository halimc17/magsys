Step by step to view FFB Receipt Report :

Get in to the menu, there are 4 report form, that are FFB Receipt Report, FFB Receipt per supplier report , FFB Receipt Report/Afdeling/Date, and FFB Receipt Report/Afdeling/Month.

  1. FFB Receipt Report
    1. Select mill and date, then click Preview to display the report.

      This report can be opened in PDF and Excel format.

    2. Here is the display of FFB Receipt Report:

  2. FFB Receipt per supplier Report
    1. Select Mill, Date, FFB, Unit/Supplier and Display then click Preview button to display the report.

      2. This report can be opened in PDF and Excel format by click PDF or Excel button above.

    2. Here is the display of default FFB Receipt per supplier Report :

      3. Here is the display of daily FFB Receipt per suppier Report :

  3. FFB Receipt Report / Afdeling / Date
    1. Select Mill, Date, Unit, and Afdeling then click Preview to display the report.

      2. This report can be opened in PDF and Excel format by click PDF or Excel button above.

    2. Here is the display of FFB Receipt Report / Afdeling / Date:

  4. FFB Receipt Report / Afdeling / Month
    1. Select mill, unit and period then click Preview to display the report.

      2. This report can be opened in PDF and Excel format by click PDF or Excel button above.
