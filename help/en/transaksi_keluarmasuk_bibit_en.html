Step by step to input Seed Movement Transaction

Get in to the menu, then 6 tabs will be appear as below :

  1. Seeding Date
    1. Please click Seeding Date tab

      2. Complete the data above then click save button, to saved or click the cancel button to cancelled.

      There is a facilities to finding all of data that has been saved by input the date, batch code, and select the status then click find button, the data will be come out automatically

      Here is the result of searching data that has been to input

    2. Data that has been saved will be appear at table data saved :

      3. Transaction Code is TMB = Tanam Bibit.

    3. Click this icon to edit a Seeding Date
    4. Click this icon to delete a Seeding Date
    5. Click this icon to posting a Seeding Date
  2. Doubletoon Tab
    1. Click Doubletoon Tab

      Double toon is Seed that have grown by more than 1 point such as 2,3 or 4 points grows.

      Batch code that selected at doubletoon must already done posting on the Seeding date tab.

      Please complete data as above then click save button to save data or click cancel button to cancelled

    2. There are facilities to finding data that has been saved, by input date, batch code, and status then data will be appear at Table data.

      3. Transaction code DBT = Doubletoon

    3. Click icon to edit doubletoon data.
    4. Click icon to delete doubletoon data.
    5. Click icon to posting doubletoon data.
  3. Reject Tab
    1. Click Reject Tab

      2. Reject is seeds not perfect

      The selected Batch at reject must already posted before at Seeding tab.

      Complete data as above then click save button to saved data, or click cancel button to cancelled.

    2. There is the same facilities as before to find the data that has been saved.

    3. Data that has been saved will be appear at table data :

      4. Transaction Code AFB = Reject

    4. Click icon to edit reject data.
    5. Click icon to delete reject data.
    6. Click icon to posting reject data.
  4. Transplanting Tab
    1. Click Transplanting Tab

      2. Transplanting is great planted in Polybag.

      The selected Batch at transplanting must be posted before at seeding tab.

      Complete data as above than click save button to saved data, or click cancel button to cancelled.

    2. There is the same facilities to find the data that has been saved.

      3. Transaction Code TPB = Transplanting

    3. Click icon to edit transplanting data.
    4. Click icon to delete transplanting data.
    5. Click icon to posting transplanting data.
  5. Seed Delivery Tab
    1. Click Seed Delivery Tab

      2. The selected Batch at Seed Delivery must be posted before at seeding tab.

      Complete data as above than click save button to saved data, or click cancel button to cancelled.

    2. Data that has been saved will be appear at Saved data form :

      3. Transaction code PNB = Seed Delivery

    3. Click icon to delete seed delivery data.
    4. Click icon to posting seed delivery data. After posting, data can not deleted
    5. Click icon to view seed delivery data in PDF format.
