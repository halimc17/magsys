QC REPORT

Module : Estate
About : QC Report
Contents:

Get in to the menu, there are 3 tabs, QC Appraisal (Area) Tab, QC Appraisal (Employee) Tab, and QC data Recap Tab as shown below :

  1. QC Appraisal (Area) Tab
    1. Click QC Appraisal (Area) Tab, select unit, afdeling, block, date and QC type then click Check Value button to display the report, or click Excel button to display in Excel format.

  2. QC Appraisal (Employee) Tab
    1. Click QC Appraisal (Employee) Tab, select unit, employee, date and QC type then click Check Value button to display the report or click Excel button to view in Excel format.

  3. QC Data Recap Tab
    1. Click QC Data Recap, select unit, QC type and date, then click Preview button to display the report or click Excel button to view in Excel format.
