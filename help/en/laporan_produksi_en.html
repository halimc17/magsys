Step by step to view Production Report :

Get in to the menu, there are 3 tabs as shown below :

  1. Estate Production Report Tab
    1. Select period, FFB (External, Internal, Afiliation), unit/supplier, then click Preview button.

      2. This report can be opened in PDF and Excel format by click PDF or Spreadsheet button as above.

    2. Here is the display of Estate Production Report :

  2. Estate Production Detail Tab
    1. Click Estate Production Detail tab. Select period and unit then click Preview button to display the report.

      2. This report can be opened in PDF and Excel format by click PDF or Spreadsheet button as above.

    2. Here is the display of Estate Production Detail :

  3. Production Trend Tab
    1. Select period and unit, then click Preview button to display the report in HTML format or click Spreadsheet button to display the report in Excel format.
