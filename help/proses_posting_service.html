PROSES POSTING SERVICE

Modul : Traksi
Tentang : Proses Posting Service
Isi :

Langkah-langkah Proses Posting Service :

  1. Masuk ke menu maka akan muncul list penggantian komponen yang belum posting.
  2. Untuk mempermudah posting service tertentu, lakukan pencarian service seperti di bawah ini :

    3. Pencarian ini berdasarkan pada no.transaksi service dan/atau tanggal. Klik tombol Cari untuk melakukan pencarian.

  3. Berikut hasil pencariannya :

  4. Klik link Blm Posting pada kolom Action untuk melakukan posting service. Akan muncul konfrimasi seperti di bawah ini :

    5. Klik tombol OK untuk melakukan posting, data yang sudah diposting tidak ditampilkan pada List Data, yang hanya ditampilkan adalah List data yang belum posting

  5. Klik icon untuk melihat laporan penggunaan komponen dalam format PDF
  6. Klik icon untuk melihat list data penggantian komponen
