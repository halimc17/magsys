SETUP STANDARD NILAI KEGIATAN

Modul : IT
Tentang : Setup Standard Nilai Kegiatan
Isi :

Langkah-langkah Setup Standard Nilai Kegiatan :

  1. Masuk ke menu, lengkapi data seperti di bawah ini kemudian klik tombol Simpan :

    2. Sangat Baik, nilai 1=max 0% dari waktu penyelesaian dikurang standar waktu penyelesaian.
    Baik, nilai 2=max 20% dari waktu penyelesaian dikurang standar waktu penyelesaian.
    Cukup, max 3=max 40% dari waktu penyelesaian dikurang standar waktu penyelesaian.
    Kurang = nilai 4, diatasnya berarti kurang.

  2. Data yang sudah tersimpan akan muncul pada List Data di bawah ini :

  3. Klik icon untuk edit data standard nilai kegiatan.
  4. Klik icon untuk hapus data standard nilai kegiatan.
