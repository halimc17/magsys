HASIL TIMBANG TBS KE PIHAK EKSTERNAL

Modul :Transaksi
Tentang : Hasil Timbang TBS ke Pihak Eksternal
Isi :

Langkah-langkah Transaksi Hasil Timbang TBS ke Pihak Eksternal :

  1. Masuk ke menu, klik tombol buat baru lengkapi form entry data seperti dibawah ini

  2. Klik tombol simpan ketika sudah selesai mengisi data
  3. Data yang sudah tersimpan akan muncul pada List data dibawah ini

  4. Klik icon untuk melihat data dalam format PDF.
  5. Terdapat aplikasi untuk mencari data berdasarkan SPAT no dan tanggal timbang
