SETUP MATA UANG DAN KURS

Modul : Setup
Tentang : Setup Mata Uang dan Kurs
Isi :

Langkah-langkah Setup Mata Uang dan Kurs :

  1. Masuk ke menu, isikan kode jurnal, mata uang, simbol dan kode ISO klik pada kolom kosong untuk menambahkan setup mata uang dan kurs
  2. Lalu klik icon untuk melakukan penyimpanan data setup mata uang
  3. Klik icon untuk edit data header mata uang.
  4. Klik icon untuk hapus data header mata uang.

  5. Klik icon untuk menampilkan detail mata uang.

    Detail mata uang yang dipilih dapat di filter berdasarkan periode yang dipilih

  6. Klik icon untuk hapus data detail mata uang.
