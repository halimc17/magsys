TRANSAKSI SURAT PENGANTAR ANGKUT TBS

Modul : Kebun
Tentang : Transaksi Surat Pengantar Angkut TBS
Isi :

Langkah-langkah Transaksi Surat Pengantar Angkut TBS :

  1. Masuk ke menu, klik icon untuk input surat pengantar angkut TBS, berikut tampilan form nya :

    2. Lengkapi data seperti di atas kemudian klik tombol Simpan.

  2. Setelah Data Kop disimpan maka akan muncul form rincian seperti di bawah ini :

    3. Untuk BJR dikunci mengambil data dari BJR blok sesuai yang dipilih, BJR Blok bulan sebelumnya. Lengkapi data seperti di atas kemudian klik icon untuk simpan rincian.

    Jika sudah selesai input rincian SPAT maka klik tombol Selesai.

  3. Data Rincian yang sudah disimpan akan muncul pada Data Tersimpan di bawah ini :

  4. Klik icon untuk edit rincian data.
  5. Klik icon untuk delete rincian data.
  6. Klik icon untuk meihat semua list data SPAT atau lakukan pencarian data seperti di bawah ini :

  7. Berikut hasil pencariannya :

  8. Klik icon untuk edit data SPAT
  9. Klik icon untuk delete data SPAT
  10. Klik icon untuk lihat data SPB dalam format PDF.
  11. List Data SPAT dapat dibuka dalam format Excel dan PDF dengan memilih periode dan kode organisasi pada form Export Data seperti di bawah ini :

    12. Klik icon untuk melihat list data SPB dalam format Excel atau klik icon untuk melihat list data SPB dalam format PDF.
