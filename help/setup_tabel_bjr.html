Langkah-langkah Setup Tabel BJR & Kelas Pohon :

  1. Masuk ke menu maka akan muncul form seperti di bawah ini :

    2. List Data di atas menunjukkan bahwa terdapat 2 tahun produksi yaitu tahun produksi 2008 dan 2009.

  2. Masukkan tahun produksi kemudian klik tombol Ok.

  3. Untuk input data BJR, lengkapi data seperti di bawah ini :

    4. Klik tombol Simpan untuk menyimpan data

  4. Data yang sudah disimpan akan muncul pada List Data
  5. Klik icon untuk edit data tabel BJR.
  6. Klik icon untuk hapus data tabel BJR.
  7. Sebagai catatan untuk yang dapat mengedit dan menghapus Setup Tabel BJR & Kelas Pohon ini adalah yang berlokasi tugas di HO, jika di kebun maka fitur edit dan delete tidak ditampilkan
