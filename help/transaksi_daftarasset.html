TRANSAKSI DAFTAR ASSET

Modul : Keuangan
Tentang : Transaksi Daftar Asset
Isi :

Langkah-langkah Transaksi Daftar Asset :

Masuk ke menu maka akan terdapat 2 tab yaitu tab Form Input Asset dan tab List Data.

  1. Tab Form Input Asset
    1. Klik tab Form Input Asset atau klik icon untuk input data asset. Berikut tampilan form nya :

      2. Lengkapi data atau klik tombol Batal untuk membatalkan input data asset.

    2. Untuk pengisian No.Dok Pengadaan & No. Induk dapat melakukan pencarian dengan klik icon maka akan muncul form pencarian

      Tampilan diatas adalah form untuk pencarian No.Dok Pengadaan dapat dipilih Document berdasarkan PO atau SPK dengan memilih panah pada field Document lalu isi No PO lalu klik Go, klik pada No Dokumen yang dimaksud :

      Tampilan diatas adalah form pencarian untuk No. Induk asset dengan mengetikkan No yang ingin dicari lalu klik tombol Go maka data yang dicari akan muncul, klik pada No Induk Asset yang dimaksud maka data akan muncul pada form header

      Setelah selesai mengisi form input daftar asset klik tombol simpan pada form

  2. Tab List Data
    1. Klik tab List Data atau klik icon untuk melihat semua list data asset.

    2. Dapat dilakukan pencarian untuk list data asset dengan pencarian berdasarkan Nama Asset atau kode asset
      3. Berikut tampilannya :

      Klik tombol cari untuk melakukan pencarian.

    3. Berikut hasil pencariannya :

    4. Klik icon untuk edit data asset.
