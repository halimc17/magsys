PEMAKAIAN BARANG

Modul : Pengadaan
Tentang : Pemakaian Barang
Isi :

Langkah-langkah pemakaian barang:

Masuk ke menu, akan terdapat form daftar gudang dan 2 tab yaitu tab Pengeluaran Barang dan List Data.

  1. Form Daftar Gudang

    2. Pilih gudang yang dimaksud, Pemilik Barang akan otomatis muncul kemudian klik Simpan. Tombol Ganti digunakan untuk mengganti pilihan gudang.

  2. Tab Pengeluaran Barang
    1. Setelah klik tombol Simpan di atas maka secara otomatis No. Dokumen pengeluaran barang akan terbentuk kemudian isikan Untuk, Penerima, Sub Unit, Catatan dan rincian barang yang dipakai seperti di bawah ini :

      2. Untuk memperoleh kode barang pada rincian, klik pada box kode barang maka akan muncul window baru untuk melakukan pencarian barang kemudian klik pada barang yang dimaksud. Barang yang akan dipakai harus memiliki saldo, jika saldo nol maka barang tidak bisa dipilih.

    2. Berikut tampilan setelah memilih barang di atas :

      3. Lengkapi data Jumlah, Blok, Mesin dan Kegiatan. Jika rincian data sudah lengkap maka klik tombol Simpan.

    3. Berikut tampilan data untuk rincian data yang sudah tersimpan :

      4. Jika masih ada barang lain yang diperlukan, bisa diisikan kembali pada form Rincian di atas.

    4. Klik tombol selesai jika barang yang dimaksud sudah selesai diisikan pada form rincian
  3. Tab List Data
    1. Klik Tab List Data maka akan muncul semua transaksi pemakaian barang di gudang. Lakukan pencarian untuk transaksi pemakaian barang yang dimaksud seperti tampilan di bawah ini :

    2. Berikut hasil pencarian transaksi di atas :

    3. Klik icon untuk melakukan pemakaian barang kembali
    4. Klik icon untuk hapus transaksi pemakaian barang.
    5. Klik icon untuk melihat Berita Acara Serah Terima Barang dalam format PDF. Status penerimaan belum posting, selanjutnya dapat melakukan langkah posting pada menu administrasi posting
