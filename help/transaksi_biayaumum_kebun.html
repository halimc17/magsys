TRANSAKSI BIAYA UMUM KEBUN

Modul : Anggaran
Tentang : Transaksi Biaya Umum Kebun
Isi :

Langkah-langkah input Transaksi Biaya Umum Kebun :

Masuk ke menu, akan terdapat form biaya umum kebun dan 3 Tab yaitu Tab List Data, Tab Sebaran dan Tab Tutup seperti di bawah ini :

  1. Form Biaya Umum
    1. Lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data biaya umum kebun.

    2. Akan muncul pesan berikut :

      3. Klik tombol OK.

  2. Tab List Data
    1. Data yang sudah tersimpan akan muncul pada Tab List Data berikut ini :

    2. Klik icon untuk hapus data biaya umum kebun.
  3. Tab Sebaran
    1. Klik Tab Sebaran, akan muncul semua sebaran budget biaya umum kebun :

    2. Klik icon untuk mengisi sebaran, berikut tampilan form nya :

      3. Sesuaikan prediksi persen dengan budget, isikan prediksi persen nya saja, nominal akan dihitung oleh sistem.

    3. Klik icon untuk simpan data sebaran.
    4. Klik icon untuk mengosongkan data sebaran.
    5. Data sebaran sudah terisi :

  4. Tab Tutup
    1. Klik Tab Tutup untuk melakukan finalisasi budget, pilih tahun budget kemudian klik tombol List Data untuk menampilkan List Data budget biaya umum.

    2. Klik tombol Tutup untuk melakukan finalisasi budget, akan muncul konfirmasi berikut :

      3. Klik tombol OK jika sudah yakin tidak ada perubahan data pada budget. Konfirmasi ini akan muncul pada setiap baris data budget di atas.

      Jika semua data budget sudah tutup maka akan muncul pesan di bawah ini :

      Klik tombol OK.
