Tentang : Laporan Data Karyawan
Isi :

Langkah-langkah melihat Laporan Data Karyawan :

  1. Masuk ke menu, terdapat 2 pencarian :
    1. Pencarian berdasarkan nama karyawan, lokasi tugas, tipe karyawan, status dan jenis kelamin.
    2. Pencarian berdasarkan tahun masuk, bulan masuk, tahun keluar dan bulan keluar.

  2. Klik icon untuk melihat data karyawan hasil dari pencarian di atas dalam format Excel.
