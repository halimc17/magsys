SETUP POTONGAN BPJS

Modul : SDM
Tentang : Setup Potongan BPJS
Isi :

Langkah-langkah Setup Pot BPJS:

  1. Masuk ke menu SDM - Setup - Pot BPJS, isikan porsi BPJS untuk karyawan, perusahaan dan Max Gaji seperti di bawah ini :

    2. Keterangan :

    - Lokasi Tugas = Lokasi Karyawan ( PABRIK / KEBUN )

    - jenis BPJS = ( BPJS Kesehatan / BPJS Ketenagakerjaan )

    - Beban Karyawan % = Adalah persen yang menjadi tanggungan karyawan dan akan dipotong dari gaji karyawan

    - Beban Perusahaan % = Adalah persen yang menjadi tanggungan perusahaan

    - Max Gaji (Rp) = Adalah Max Gaji Karyawan yang akan dipotong BPJS contoh : Jika Max Gaji Rp 3.000.000,- jika karyawan memiliki gaji lebih dari Rp. 3.000.000,- maka potongan dikalikan dengan Rp. 3.000.000,- namun jika gaji karyawan kurang dari Rp. 3.000.000,- maka potongan dikalikan dengan gaji karyawan tersebut.
