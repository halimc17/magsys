RETUR KE SUPPLIER

Modul : Pengadaan
Tentang : Retur ke Supplier
Isi :

Langkah-langkah Retur ke Supplier :

Masuk ke menu, akan terdapat form daftar gudang dan 2 tab yaitu tab Pengembalian/retur dan List Data.

  1. Form Daftar Gudang

    2. Pilih gudang yang dimaksud kemudian klik Simpan. Tombol Ganti digunakan untuk mengganti pilihan gudang.

  2. Tab Penerimaan Barang
    1. Setelah klik tombol Simpan di atas maka secara otomatis No. Dokumen retur ke supplier akan terbentuk kemudian isikan No. Dokumen Lama dan Kode Barang kemudian klik tombol Check untuk menampilkan detail dokumen lama seperti di bawah ini :

      2. Isikan kuantitas retur dan keterangan kemudian klik tombol Simpan.

  3. Tab List Data
    1. Klik Tab List Data maka akan muncul semua transaksi retur ke supplier. Lakukan pencarian untuk transaksi retur ke supplier yang dimaksud seperti tampilan di bawah ini :

    2. Klik icon untuk menghapus transaksi retur ke supplier jika statusnya masih belum posting.
    3. Klik icon untuk cetak dokumen retur barang dalam format PDF . Karena retur ini masih belum diposting maka status bukti penerimaan barang ini masih "Not Confirm".
