TRANSAKSI RENCANA KERJA BULANAN

Modul : Kebun
Tentang : Input Rencana Kerja Bulanan (RKB)
Isi :

Langkah-langkah Input RKB :

  1. Input RKB (Rencana Kerja Bulanan) dilakukan melalui menu: Kebun-Transaksi-Rencana Kerja Bulanan. :

  2. Inputkan Header yang terdiri dari Periode (Periode Kerja), Kebun, Manager, Asisten, Kode Blok, Kode Kegiatan dan Volume Kegiatan. Klik Simpan untuk melanjutkan. Isian Manager akan menentukan siapa yang dapat mem-posting inputan RKB ini.

  3. Pada inputan detail ada lima tab: HK, Material, Kontrak, Kendaraan/Alat Berat dan Peralatan.

  4. Pada inputan detail HK, silakan mengisi jumlah HK KBL, KHT, KNT dan KHL (total HK akan terisi otomatis). Isikan pula nilai Rupiah Upah dan Rupiah Premi untuk keseluruhan HK. Klik Simpan untuk menyimpan, logo Edit untuk mengganti nilai yang sudah tersimpan dan logo Delete untuk menghapus data.

  5. Pada inputan detail Material, silakan mengisikan nama barang/kode barang kemudian meng-klik logo Cari untuk mencari barang. Klik pada barang yang diinginkan (ditampilkan pula harga rata-rata barang tersebut). Isikan jumlah barang, Rupiah Material akan otomatis terisi

  6. Pada inputan detail Kontrak, isikan nomor SPK kemudian klik logo Cari untuk mencari SPK. SPK harus terlebih dahulu diinputkan di OWL. Hanya akan muncul SPK yang sesuai dengan informasi pada header (kode kegiatan dan kode blok)

  7. Pada inputan detail Kendaraan/Alat Berat, pilih kode kendaraan/alat berat. Sistem akan otomatis menghitung Rupiah per HM/KM berdasarkan data periode kerja -2.

  8. Pada inputan detail Peralatan, tampilannya mirip dengan inputan detail Material tanpa harga. Data ini hanya digunakan sebagai informasi alat yang digunakan.

  9. Setelah diinput maka akan muncul data headernya pada list, untuk menginput data kegiatan/blok selanjutnya silakan menginput headernya kembali.
