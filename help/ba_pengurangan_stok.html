BA PENGURANGAN STOK

Modul : Pabrik
Tentang : Transaksi BA Pengurangan Stok
Isi :

Langkah-langkah input BA Pengurangan Stok :

  1. Masuk ke menu, akan terdapat 2 tab yaitu form dan laporan seperti di bawah ini :

    2. Untuk tab form lengkapi data kemudian klik tombol simpan untuk menyimpan data

  2. Data yang sudah tersimpan akan muncul pada List Data disamping form inputan

  3. Terdapat aplikasi untuk mencari data berdasarkan nama barang dan tanggal

  4. Klik tab Laporan untuk melihat laporan seperti dibawah ini lengkapi data yang dibutuhkan untuk menampilkan Laporan BA Pengurangan Stok Pabrik :

    5. Klik tombol Preview untuk menampilkan data seperti dibawah ini

    Laporan ini dapat juga dibuka dalam format Excel dengan klik tombol Excel diatas

  5. Klik icon untuk mengedit data BA Pengurangan Stok Pabrik
  6. Klik icon untuk menghapus data BA Pengurangan Stok Pabrik
