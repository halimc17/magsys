Modul : Pengadaan
Tentang : Pengiriman Internal
Isi :

Langkah-langkah Pengiriman Internal :

  1. Masuk ke menu maka akan muncul list job per PT seperti di bawah ini :

    2. Klik pada PT yang dimaksud maka akan muncul semua transaksi penerimaan barang supplier.

  2. Berikut tampilan semua list penerimaan barang supplier :

    3. Tick pada kolom Action untuk no transaksi penerimaan barang yang akan dilakukan pengiriman internal kemudian klik tombol Proses.

  3. Setelah klik tombol Proses maka No. Surat Jalan akan keluar secara otomatis, berikut tampilannya :

  4. Lengkapi data di atas seperti di bawah ini :

    5. Untuk Expeditor/ Intenal dapat pilih langsung dari drop down menu atau dengan klik icon , berikut tampilannya :

    Ketikkan nama expeditor/internal kemudian klik tombol Cari maka akan muncul hasil pencarian. Pilih pada expeditor yang dimaksud.

  5. Setelah data sudah selesai diisi, klik tombol Simpan maka transaksi pengiriman internal ini akan muncul pada List Data seperti di bawah ini :

    6. Atau dapat dilakukan pencarian untuk transaksi tersebut sebagai berikut :

    Isikan No. Surat Jalan kemudian klik tombol Cari.

    Jika ingin meihat semua list transaksi pengiriman internal, klik icon

  6. Klik icon untuk edit surat jalan.
  7. Klik icon untuk hapus surat jalan.
  8. Klik icon untuk melihat Surat Jalan dalam format PDF seperti di bawah ini :
