PENJUALAN TBS

Modul : Kebun
Tentang : Penjualan TBS ke pihak luar
Isi :

Langkah-langkah input Transaksi Penjualan TBS, menu ini adalah menu untuk menampung kebutuhan informasi TBS ke pihak luar :

  1. Masuk ke menu, akan terdapat form pengisian untuk data penjualan TBS ke pihak luar :

    2. Lengkapi data kemudian klik tombol simpan

  2. Data yang sudah tersimpan akan muncul pada list data dibawah ini :

  3. Klik icon edit untuk melakukan edit data Penjualan TBS
  4. Klik icon delete untuk menghapus data Penjualan TBS
