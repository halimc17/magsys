TRANSAKSI PERTANGGUNGJAWABAN PERJALANAN DINAS

Modul : SDM
Tentang : Transaksi Pertanggungjawaban Perjalanan Dinas
Isi :

Langkah-langkah input Transaksi Pertanggungjawaban Perjalanan Dinas :

Masuk ke menu, akan terdapat 3 Tab yaitu Tab Form, Tab Hasil kerja dan Tab List Data seperti di bawah ini :

  1. Tab Hasil Kerja
    1. Klik Hasil Kerja,
    2. isikan uraian hasil perjalanan dinas kemudian klik tombol Simpan.

      Klik icon untuk melihat Surat Perjalanan Dinas dalam format PDF. Klik tombol Simpan untuk menyimpan rincian pertanggungjawaban perjalanan dinas.

    3. Akan muncul konfirmasi seperti di bawah ini :

      4. Klik tombol OK untuk melanjutkan proses. Akan muncul pesan seperti di bawah ini :

      Klik tombol OK.

    4. Klik icon untuk hapus rincian pertanggungjawaban perjalanan dinas.
  2. Tab Form
    1. Klik Tab Form, lengkapi data seperti dibawah ini :

    2. Akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK untuk melanjutkan proses. Akan muncul pesan seperti di bawah ini :

      Klik tombol OK.

    3. Rincian Pertanggungjawaban Perjalanan Dinas yang sudah tersimpan akan muncul pada form Data Tersimpan seperti di bawah ini :

      4. Jika sudah selesai mengisikan rincian, klik tombol Selesai.

  3. Tab List Data
    1. Klik Tab List Data, akan muncul semua list data pertanggungjawaban perjalanan dinas.

      2. Terdapat fasilitas pencarian berdasarkan no.transaksi

    2. Klik icon yang pertama untuk melihat biaya pada surat perjalanan dinas
    3. Klik icon yang kedua untuk melihat uraian hasil tugas pada surat perjalanan dinas
    4. Klik icon untuk edit rincian pertanggungjawaban perjalanan dinas.
