SETUP KELOMPOK SUPPLIER/KONTRAKTOR

Modul : Pengadaan
Tentang : Setup Kelompok Supplier/Kontraktor
Isi :

Langkah-langkah Setup Kelompok Supplier/Kontraktor :

  1. Masuk ke menu, lengkapi data seperti di bawah ini :

    2. Pilih tipe Supplier, Kontraktor, atau Transportir kode jurnal akan terbentuk secara otomatis oleh sistem, isi nama kelompok dan pilih nomor akun kemudian klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan penyimpanan data.

  2. Data kelompok supplier/kontraktor akan ditampilkan pada List Data.

    3. Jika pilih tipe Supplier maka list data supplier akan ditampilkan :

    Jika pilih tipe Kontraktor maka list data kontraktor akan ditampilkan :

    Jika pilih tipe Transportir maka list data transportir akan ditampilkan :

  3. Klik icon untuk edit kelompok supplier/kontraktor.
  4. Klik icon untuk hapus kelompok supplier/kontraktor.
