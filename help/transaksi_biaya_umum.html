TRANSAKSI BIAYA UMUM

Modul : Anggaran
Tentang : Transaksi Biaya Umum
Isi :

Langkah-langkah input Transaksi Biaya Umum :

Masuk ke menu, akan terdapat Form Biaya Umum dan 3 Tab yaitu Tab List Data, Tab Sebaran dan Tab Tutup seperti di bawah ini :

  1. Form Biaya Umum
    1. Lengkapi data seperti di bawah ini kemudian klik tombol Simpan :

      2. Akan muncul pesan berikut :

      Klik tombol OK.

    2. Jika jenis biaya yang digunakan memiliki Alat Berat/ Kendaraan maka isikan Jam/Thn, perhitungan Jumlah per Tahun akan dilakukan oleh sistem seperti contoh di bawah ini :

  2. Tab List Data
    1. Data yang sudah tersimpan akan muncul pada Tab List Data berikut :

    2. Klik icon untuk hapus budget biaya umum.
  3. Tab Sebaran
    1. Klik Tab Sebaran, akan muncul semua sebaran budget biaya umum :

    2. Klik icon untuk melihat detail sebaran seperti di bawah ini :

      3. Prediksi % diperoleh dari 100% / 12 bulan, sedangkan sebaran rupiah diperoleh dari Total Biaya / 12.

    3. Cek data sebaran tersebut, jika sudah sesuai klik icon untuk menyimpan data sebaran, akan muncul pesan berikut :

      4. Klik tombol OK.

    4. Klik icon jika akan mengosongkan form, akan muncul konfirmasi berikut :

      5. Klik OK untuk melanjutkan proses.

    5. Data Sebaran sudah tersimpan :

  4. Tab Tutup
    1. Klik Tab Tutup untuk melakukan finalisasi budget, pilih tahun budget kemudian klik tombol List Data untuk menampilkan List Data budget biaya umum.

    2. Klik tombol Tutup untuk melakukan finalisasi budget, akan muncul konfirmasi berikut :

      3. Klik tombol OK jika sudah yakin tidak ada perubahan data pada budget. Konfirmasi ini akan muncul pada setiap baris data budget di atas.

      Jika semua data budget sudah tutup maka akan muncul pesan di bawah ini :

      Klik tombol OK.
