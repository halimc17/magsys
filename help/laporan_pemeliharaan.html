LAPORAN PEMELIHARAAN

Modul : Kebun
Tentang : Laporan Pemeliharaan
Isi :

Langkah-langkah Laporan Pemeliharaan :

Masuk ke menu, akan terdapat 2 tab yaitu Pemeliharaan dan Rotasi Pemeliharaan, berikut tampilannya :

  1. Pemeliharaan
    1. Pilih kebun, afdeling, tanggal dari dan tanggal sampai dengan, kegiatan kemudian klik tombol Preview untuk menampilkan Laporan.

      2. Laporan ini dapat dilihat dalam format Excel dengan klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Pemeliharaan :

  2. Rotasi Pemeliharaan
    1. Pilih kebun, afdeling, tahun dan kegiatan kemudian klik tombol Preview untuk menampilkan Laporan.

      2. Laporan ini dapat ditampilkan dalam format Excel dengan klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Pemeliharaan :

    3. Klik pada jumlah HK atau hasil kerja atau output untuk menampilkan detail pekerjaan. Berikut tampilan untuk detail pekerjaan :
