Langkah-langkah Setup Pendidikan :

  2. Data yang sudah tersimpan akan muncul pada form Tingkat Pendidikan Yang Tersedia di bawah ini :

  3. Klik icon untuk edit data pendidikan.
  4. Klik icon untuk hapus data pendidikan.
