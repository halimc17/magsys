Modul : Kebun
Tentang : Historis Aresta
Isi :

Langkah-langkah melihat historis aresta :

  1. Masuk ke menu, pilih kebun kemdian pilih tahun menyesuaikan kebutuhan Medco dapat filter Laporan historis Aresta per tahun.
  2. Lalu klik tombol Preview untuk menampilkan Laporan dalam format HTML atau klik tombol Excel untuk menampilkan Laporan dalam format Excel.

  3. Berikut tampilan dari Historis Aresta :

    4. Klik pada kolom mutasi untuk menampilkan detail mutasi.
