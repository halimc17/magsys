Tentang : Transaksi Service
Isi :

Langkah-langkah input Transaksi Service :

  1. Masuk ke menu, klik icon untuk input transaksi service. Berikut tampilan form Kop transaksi service :

    2. No. Transaksi akan terbentuk secara otomatis oleh sistem. Lengkapi data seperti di atas, untuk waktu perbaikan terbentuk secara otomatis by inputan tanggal jam mulai dan tanggal jam selesai kemudian klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan.

  2. Berikut tampilan form penambahan kode barang :

  3. Klik icon untuk melakukan pencarian kode barang, berikut tampilan form pencariannya :

    4. Isikan nama barang yang akan dicari kemudian klik tombol Find. Klik barang yang dimaksud pada hasil pencarian.

  4. Kemudian pada form penambahan kode barang, isikan jumlah dan keteranganya seperti di bawah ini :

  5. Klik icon untuk menyimpan barang. Berikut tampilan barang yang sudah disimpan :

  6. Klik icon untuk edit detail barang.
  7. Klik icon untuk hapus detail barang

    8. Klik icon untuk melihat list data service.

  8. Terdapat fasilitas pencarian seperti di bawah ini :

    9. Pencarian ini berdasarkan no. transaksi dan/atau tanggal. Kli tombol Cari untuk melakukan pencarian

  9. Berikut hasil pencariannya :

  10. Klik icon untuk edit data service.
  11. Klik icon untuk hapus data service.
