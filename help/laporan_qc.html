LAPORAN QC

Modul : Kebun
Tentang : Laporan QC
Isi :

Masuk ke menu, akan terdapat 3 Tab yaitu Tab Penilaian QC(Areal), Tab Penilaian QC(Karyawan), dan Tab Rekap Data QC seperti di bawah ini :

  1. Penilaian QC(Areal)
    1. Klik Tab Penilaian QC(Areal), pilih unit, afdeling, blok, tanggal dan tipe QC kemudian klik tombol CekNilai untuk menampilkan laporan atau klik tombol Excel untuk menampilkan laporan dalam format Excel.

  2. Penilaian QC(Karyawan)
    1. Klik Tab Penilaian QC(Karyawan), pilih unit, karyawan, tanggal dan tipe QC kemudian klik tombol CekNilai untuk menampilkan laporan atau klik tombol Excel untuk menampilkan laporan dalam format Excel.

  3. Rekap Data QC
    1. Klik Tab Rekap Data QC, pilih unit, tipe QC dan tanggal kemudian klik tombol Preview untuk menampilkan laporan atau klik tombol Excel untuk menampilkan laporan dalam format Excel.
