SURAT JALAN

Modul : Pengadaan
Tentang : Pengiriman Internal Surat Jalan
Isi :

Langkah-langkah Transaksi Pengiriman Internal Surat Jalan :

  1. Masuk ke menu, akan terdapat form inputan seperti gambar dibawah ini klik buat baru untuk menginput transaksi Surat Jalan :

    2. Lengkapi data yang dibutuhkan nomor surat jalan akan terbentuk secara otomatis dan kode organisasi di disable berdasarkan lokasi tugas pembuat transaksi surat jalan tersebut

  2. Setelah input transaksi Surat Jalan klik tombol simpan seperti gambar dibawah ini :

  3. Untuk mendapatkan detail dapat mengambil data dari PO, Package List, dan Material List dengan klik tombol masing-masing data detail tersebut

  4. Berikut tampilan detail untuk data berdasarkan No. PO, klik tombol Add Detail From PO akan muncul form masukkan No.PO atau tahun klik No.PO yang dimaksud setelah itu Add detail maka data akan masuk secara otomatis ke dalam tabel detail :

  5. Lakukan hal yang sama jika ingin menambahkan detail untuk Package List dan Material List dengan klik tombol detail yang ingin ditambahkan, input No. dokumen lakukan pencarian lalu add detail.

  6. Setelah add detail selanjutnya pada List Detail isi kolom jumlah, maka tampilan kolomnya secara otomatis akan menjadi seperti gambar dibawah ini, klik tombol simpan apabila sudah selesai input jumlah :

  7. Klik List data untuk melihat List Transaksi Surat Jalan yang sebelumnya sudah diinput, selanjutnya lakukan posting dengan mengisi tanggal tiba terlebih dahulu lalu klik tombol posting bulat berwarna merah pastikan sebelum klik tombol posting dokumen lengkap :

  8. Berikut tampilan List data kolom setelah klik tombol posting tombol bulat yang sebelumnya berwarna merah akan berubah menjadi warna hijau dan secara otomatis Posting by yang sebelumnya kosong akan mengisi nama User yang membuat transaksi Surat Jalan tersebut
