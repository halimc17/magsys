Tentang : Project Monitoring/Management
Isi :

Kesulitan dalam memonitor nilai aktiva dalam konstruksi sudah dapat dihilangkan dengan modul ini. Tujuan utama modul ini adalah untuk mencatat secara tepat nilai sebuah project aktiva dari awal sampai project selesai dan dipindah ke daftar aktiva.

Contoh:
Sebuah project perumahan staff akan dibangun 25 unit. Project akan melibatkan banyak pihak antara lain: Kontraktor, Karyawan harian, Traksi, Gudang dan bagian terkait lainnya dalam perusahaan.

Bagaimana mengontrol perkembangan/penggunaan biaya project tersebut ? Modul Project Management ini adalah solusinya:
