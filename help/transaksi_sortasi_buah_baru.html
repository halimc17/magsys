TRANSAKSI SORTASI BUAH BARU

Modul : Pabrik
Tentang : Transaksi Sortasi Buah Baru
Isi :

Langkah-langkah input Transaksi Sortasi Buah Baru:

  1. Masuk ke menu, klik icon untuk input transaksi sortasi buah baru. Akan muncul Form Entry Data seperti di bawah ini :

    2. Pilih unit dan tanggal kemudian klik tombol Proses maka akan muncul form rincian.

  2. Berikut tampilan form rinciannya :

    3. Klik pada no tiket yang dimaksud kemudian isikan data pada form rincian di bawah ini kemudian simpan

    Jika rincian sudah selesai diinput, klik tombol Selesai.

  3. Data yang sudah tersimpan akan muncul pada List Data atau klik icon untuk melihat semua list data.
  4. Terdapat fasilitas pencarian no. tiket :

  5. Berikut hasil pencariannya :

    6. Klik pada salah satu kolom untuk melakukan edit data. Klik icon untuk hapus data rincian sortasi buah baru.
