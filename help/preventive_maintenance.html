PREVENTIVE MAINTENANCE

Modul : Umum
Tentang : Preventive Maintenance
Isi :

Langkah-langkah input Preventive Maintenance :

  1. Masuk ke menu, klik icon untuk input data preventive maintenance. Berikut terdapat informasi untuk cara pengisiannya :

  2. Berikut tampilan form input preventive maintenance :
    1. Contoh input jenis UMUM :

      2. Lengkapi data seperti di atas kemudian klik tombol Simpan.

      Akan muncul form detail seperti di bawah ini :

      Jika data tidak mempunyai kode barang maka tekan tombol Selesai di atas.

    2. Contoh input jenis TRAKSI :

      3. Lengkapi data seperti di atas kemudian klik tombol Simpan.

      Akan muncul form detail seperti di bawah ini :

      Klik icon untuk mencari kode barang, berikut tampilanya :

      Pilih kode barang pada hasil pencarian di atas dan isikan jumlah kemudian klik icon untuk menyimpan detail.

      Tambahkan kode barang lain jika masih ada kemudian klik tombol Selesai.

      Klik icon untuk hapus data detail.

  3. Klik icon untuk melihat semua data yang sudah tersimpan :

    4. List Data untuk jenis Umum :

    List Data untuk jenis TRAKSI :

    Klik icon untuk melihat detail data preventive maintenance.
    Klik icon untuk hapus data preventive maintenance.
    Klik icon untuk edit data preventive maintenance.
    Klik icon untuk melihat data preventive maintenance dalam format PDF.

  4. Klik icon untuk melihat semua daftar preventive maintenance yang sudah jatuh tempo. Berikut tampilan Over Due List :

    5. Preventive maintenance yang sudah selesai dikerjakan akan mempunyai status 'Selesai'. Sedangkan yang belum selesai hanya akan menampilkan check box.

    Jika preventive maintenance pada over due list ini sudah selesai dikerjakan, tick pada check box di atas maka status akan berubah menjadi 'Selesai'.
