TRANSAKSI TIMBANGAN

Modul : Pabrik
Tentang : Transaksi Timbangan
Isi :

Langkah-langkah input Transaksi Timbangan :

Masuk ke menu maka akan muncul 2 tab yaitu tab Form dan tab List Data seperti di bawah ini :

  1. Tab Form
    1. Klik Tab Form.

      2. Pilih nama barang yang akan diinput maka akan muncul form entry data.

    2. Berikut tampilan dari form entry data timbangan TBS:

      3. Lengkapi data seperti di atas kemudian klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkan.

  2. Tab List Data
    1. Klik Tab List Data

      2. Semua list data timbangan pabrik ada pada tab ini.

      Isikan no transaksi kemudian klik tombol Cari.

    2. Klik icon untuk edit data timbangan pabrik.
    3. Klik icon untuk hapus data timbangan pabrik.
    4. Klik icon untuk melihat data timbangan pabrik dalam format PDF.
