Langkah-langkah Setup Gaji Pokok Unit :

  1. Masuk ke menu, lengkapi data seperti di bawah ini kemudian simpan.

  2. Data yang sudah tersimpan akan masuk ke List Data Gaji Pokok di bawah ini :

  3. Klik icon untuk melihat list data gaji pokok unit dalam format PDF.
  4. Klik icon untuk melihat list data gaji pokok unit dalam format Excel.
  5. Klik icon untuk edit data gaji pokok karyawan unit.
  6. Klik icon untuk hapus data gaji pokok karyawan unit.
  7. Terdapat fasilitas pencarian berdasarkan tahun, tipe karyawan, nama karyawan dan ID komponen seperti di bawah ini :

  8. Terdapat fasilitas copy gaji pokok dari konfigurasi gaji tahun tertentu ke tahun tertentu seperti di bawah ini :

    9. Klik tombol Proses maka gaji pokok tahun 2014 akan ter-copy ke gaji pokok tahun 2015.
