LAPORAN PENGOBATAN KARYAWAN

Modul : SDM
Tentang : Laporan Pengobatan Karyawan
Isi :

Langkah-langkah melihat Laporan Pengobatan Karyawan yang belum dibayar :

Masuk ke menu, akan terdapat beberapa tab seperti di bawah ini :

  1. Rincian
    1. Terdapat pilihan tahun plafond, kode organisasi dan rumah sakit yang ingin ditampilkan seperti di bawah ini :

    2. Klik icon untuk menampilkan Laporan Pengobatan Karyawan dalam format Excel.
    3. Klik icon untuk menampilkan detail transaksi pengobatan karyawan seperti di bawah ini :

  2. Rank Diagnosa
    1. Pilih Tahun Plafond dan Kode Organisasi maka akan muncul rank diagnosa seperti di bawah ini :

    2. Klik icon untuk menampilkan Laporan Rank Diagnosa dalam format Excel.
  3. Rank Biaya/ Karyawan
    1. Pilih Tahun Plafond dan Kode Organisasi maka akan muncul rank biaya per karyawan seperti di bawah ini :

    2. Klik icon untuk menampilkan Laporan Rank Biaya per Karyawan dalam format Excel.
    3. Klik icon untuk menampilkan Detail Rank Biaya per Karyawan.
  4. Rank Biaya/ Diagnosa
    1. Pilih Tahun Plafond dan Kode Organisasi maka akan muncul rank biaya per diagnosa seperti di bawah ini :

    2. Klik icon untuk menampilkan Laporan Rank Biaya per Diagnosa dalam format Excel.
  5. Monthly Trend
    1. Pilih Tahun Plafond dan Kode Organisasi maka akan muncul Monthly Trend seperti di bawah ini :

      2. Laporan ini menampilkan jumlah biaya per bulan pada periode tertentu.

    2. Klik icon untuk menampilkan Laporan Monthly Trend dalam format Excel.
  6. By Cost Type
    1. Pilih Tahun Plafond dan Nama Karyawan maka akan muncul laporan seperti di bawah ini :

      2. Laporan ini menampilkan total per jenis biaya.

    2. Klik icon untuk menampilkan Laporan By Cost Type dalam format Excel.
  7. By Treatment Type
    1. Pilih Tahun Plafond dan Kode Organisasi maka akan muncul laporan seperti di bawah ini :

      2. Laporan ini menampilkan total biaya berdasarkan jenis pengobatan.

    2. Klik icon untuk menampilkan Laporan By Treatment Type dalam format Excel.
