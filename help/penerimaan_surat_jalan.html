Modul : Pengadaan
Tentang : Pengiriman Internal/Penerimaan Surat Jalan
Isi :

Langkah-langkah input Penerimaan Surat Jalan :

  1. Masuk ke menu, akan terdapat form List Data seperti dibawah ini, :

  2. Lakukan pencarian pada form cari surat jalan dengan menginput Nomor Surat Jalan seperti dibawah ini, lalu klik cari :

  3. Klik icon pada kolom aksi untuk menginput data surat jalan terhadap kondisi barang yang diterimakan oleh Purchaser
  4. Berikut tampilan form detail, lakukan penginputan data, silahkan mengisi Quantitiy barang yang diterimakan kemudian update dengan klik tombol simpan :

  5. Setelah selesai mengisi data lakukan posting untuk mengakui dan mengunci data agar tidak dapat diubah
  6. Berikut tampilan data setelah data berhasil di posting, pada tombol aksi akan berubah menjadi posting
