TRANSAKSI PERJALANAN DINAS

Modul : SDM
Tentang : Transaksi Perjalanan Dinas
Isi :

Langkah-langkah input Transaksi Perjalanan Dinas :

Masuk ke menu, akan terdapat 2 tab yaitu Tab Form dan Tab List Data seperti di bawah ini, klik buat baru untuk membuat transaksi perjalanan dinas :

  1. Tab Form
    1. Klik Tab Form, lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data perjalanan dinas.

  2. Tab List Data
    1. Klik Tab List Data, akan muncul list data perjalanan dinas. Terdapat juga fasilitas pencarian berdasarkan no. transaksi.

      2. Pengajuan perjalanan dinas ini masih belum disetujui oleh atasan, BOD (jika di HO) dan pihak HRD. Atasan dan pihak HRD masih dapat diiganti.

    2. Klik icon untuk hapus data perjalanan dinas.
    3. Klik icon untuk edit data perjalanan dinas.
    4. Klik icon untuk melihat Surat Perjalanan Dinas dalam format PDF seperti di bawah ini :
