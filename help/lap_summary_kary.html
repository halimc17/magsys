Masuk ke menu, akan terdapat 2 tab seperti di bawah ini :

  1. Summary Karyawan
    1. Pilih tanggal kemudian klik Preview untuk menampilkan Laporan dalam format HTML atau klik Excel untuk menampilkan Laporan dalam format Excel :

      2. Klik pada header untuk menampilkan detail laporan.

    2. Berikut tampilan dari Detail Summary Karyawan :

    3. Klik icon untuk menampilkan Detail Summary Karyawan dalam format Excel.
  2. Summary Karyawan Dari Gaji
    1. Pilih perusahaan, unit dan periode kemudian klik Proses untuk menampilkan Laporan Summmary Karyawan dari Gaji :

      2. Klik tombol Excel untuk menampilkan laporan dalam format Excel.
