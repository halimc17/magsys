LAPORAN PENERIMAAN TBS

Modul : Pabrik
Tentang : Laporan Penerimaan TBS
Isi :

Langkah-langkah meihat Laporan Penerimaan TBS :

Masuk ke menu, akan terdapat 4 form laporan yaitu Rekap Penerimaan TBS, Laporan Penerimaan TBS, Laporan Penerimaan TBS per Afdeling per Tanggal dan Laporan PenerimaanTBS per Afdeling per Bulan.

  1. Rekap Penerimaan TBS
    1. Pilih pabrik dan tanggal kemudian klik tombol Preview untuk menampilkan Rekap Penerimaan TBS.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

    2. Berikut tampilan Rekap Penerimaan TBS :

      3. Penerimaan TBS yang berwarna kuning menandakan bahwa ada buah dari afdeling lain.

  2. Laporan Penerimaan TBS
    1. Pada form Laporan Penerimaan TBS pilih pabrik, tanggal, TBS, Unit/Supplier dan tampilan kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

    2. Berikut tampilan default dari Laporan Penerimaan TBS :

    3. Berikut tampilan harian dari Laporan Penerimaan TBS :

  3. Laporan Penerimaan TBS/Afdeling/Tanggal
    1. Pada form Laporan Penerimaan TBS/Afdeling/Tanggal pilih pabrik, tanggal, unit, dan afdeling kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Penerimaan TBS/Afdeling/Tanggal :

  4. Laporan Penerimaan TBS/Afdeling/Bulan
    1. Pada form Laporan Penerimaan TBS/Afdeling/Bulan pilih pabrik, unit dan periode kemudian klik tombol Preview untuk menampilkan laporan.

      2. Laporan ini dapat dibuka dalam format PDF dan Excel dengan klik tombol PDF atau klik tombol Excel di atas.

    2. Berikut tampilan dari Laporan Penerimaan TBS/Afdeling/Bulan :
