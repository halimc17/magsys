Masuk ke menu, akan terdapat form navigasi, tab Header dan tab Rincian seperti di bawah ini :

  1. Form Navigasi
    1. Lokasi Tugas akan berisi sesuai dengan lokasi tugas user yang login. Pilih periode kemudian klik tombol Proses.

      2. Jika belum diproses maka saat klik tombol Lihat, tidak akan ada data yang muncul pada tab Kop.

    2. Akan muncul hak cuti semua karyawan Teluk Bayur Estate periode 2015 seperti di bawah ini :

      3. Klik tombol Simpan untuk menyimpan data.

    3. Tab Header
      1. Pilih periode kemudian klik tombol Lihat.

      2. Data akan muncul pada tab Kop seperti di bawah ini :

        3. Klik icon untuk menyimpan data Kop.

    4. Tab Rincian
      1. Klik icon untuk mengisi rincian, berikut tampilan form nya :

        2. Isikan tanggal cuti (dari tanggal dan sampai dengan tanggal), isikan banyaknya cuti yang diambil dan keterangan cuti kemudian klik tombol Simpan untuk menyimpan data rincian.

      3. Klik icon untuk hapus data rincian cuti.
