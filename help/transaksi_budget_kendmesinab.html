TRANSAKSI BUDGET KENDARAAN, MESIN, ALAT BERAT

Modul : Anggaran
Tentang : Transaksi Budget Kendaraan, Mesin, Alat Berat
Isi :

Langkah-langkah input Transaksi Budget Kendaraan, Mesin, Alat Berat :

  1. Masuk ke menu, isikan tahun budget, pilih kode traksi, kode kendaraan kemudian klik tombol Simpan.

  2. Akan muncul 5 Tab yaitu Tab SDM, Tab Material, Tab Service, Tab Biaya Lain dan Tab Tutup :

  3. Tab SDM
    1. Pada Tab SDM, pilih kode anggaran, HK Efektif akan muncul secara otomatis oleh sistem.Isikan jumlah personel kemudian klik tombol Simpan. Total biaya akan dihitung oleh sistem berdasarkan jumlah personelnya.

    2. Data yang sudah tersimpan akan muncul pada Data Tersimpan di bawah ini :

    3. Klik icon untuk hapus data SDM.
  4. Tab Material
    1. Klik Tab Material, pilih kode anggaran, kode barang dan isikan jumlah kemudian klik tombol Simpan. Total Harga akan dihitung oleh sistem secara otomatis.

    2. Untuk memilih kode barang, klik icon untuk melakukan pencarian barang seperti di bawah ini :

      3. Ketikkan nama barang yang dicari kemudian klik tombol Cari. Pilih barang yang dimaksud pada hasil pencarian.

    3. Data yang sudah tersimpan akan muncul pada Data Tersimpan di bawah ini :

    4. Klik icon untuk hapus data Material.
  5. Tab Service
    1. Klik Tab Service, pilih kode anggaran, kode workshop dan isikan jam/tahun kemudian klik tombol Simpan. Total Harga akan dihitung oleh sistem secara otomatis.

    2. Data yang sudah tersimpan akan muncul pada Data Tersimpan di bawah ini :

    3. Klik icon untuk hapus data Service.
  6. Tab Biaya Lain
    1. Klik Tab Biaya Lain, pilih kode anggaran, jenis biaya dan isikan total biaya kemudian klik tombol Simpan.

    2. Data yang sudah tersimpan akan muncul pada Data Tersimpan di bawah ini :

    3. Klik icon untuk hapus data Biaya Lain.
  7. Tab Tutup
    1. Klik Tab Tutup untuk melakukan finalisasi budget, pilih tahun budget kemudian klik tombol Tutup.

    2. Akan muncul konfirmasi seperti di bawah ini :

      3. Klik tombol OK untuk melanjutkan proses.

    3. Budget Kendaraan, Mesin, Alat Berat tahun 2012 sudah tutup :
