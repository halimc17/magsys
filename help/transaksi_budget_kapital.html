TRANSAKSI BUDGET KAPITAL

Modul : Anggaran
Tentang : Transaksi Budget Kapital
Isi :

Langkah-langkah input Transaksi Budget Kapital :

Masuk ke menu, akan terdapat form budget kapital dan 3 Tab yaitu Tab List Data, Tab Sebaran dan Tab Tutup seperti di bawah ini :

  1. Form Budget Kapital
    1. Lengkapi data seperti di bawah ini :

      2. Klik tombol Simpan untuk menyimpan data budget kapital.

  2. Tab List Data
    1. Data yang sudah tersimpan akan muncul pada Tab List Data berikut ini :

    2. Status sebaran budget kapital di atas masih Not.OK karena sebaran masih belum diisi. Klik icon untuk hapus budget kapital.
  3. Tab Sebaran
    1. Untuk mengisi sebaran budget kapital, klik pada baris List Data yang akan diisi sebarannya, berikut tampilan form sebaran :

      2. Sebaran ini akan dibagi rata per bulan, namun dapat disesuaikan dengan mengisi prediksi persen saja, nominal akan otomatis dihitung oleh sistem :

    2. Klik icon untuk simpan data sebaran.
    3. Klik icon untuk mengosongkan data sebaran.
    4. Data sebaran sudah terisi :

      5. Status sebaran sudah OK.

  4. Tab Tutup
    1. Klik Tab Tutup untuk melakukan finalisasi budget, pilih tahun budget kemudian klik tombol Tutup :

    2. Akan muncul konfirmasi berikut :

      3. Klik tombol OK untuk melanjutkan proses.
