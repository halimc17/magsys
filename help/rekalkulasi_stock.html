REKALKULASI STOCK

Modul : Pengadaan
Tentang : Penerimaan Barang Mutasi
Isi :

Langkah-langkah Penerimaan Barang Mutasi :

Masuk ke menu, akan terdapat 2 tab yaitu tab Rekalkulasi Stock dan tab Rekalkulasi Stock Tidak Normal seperti di bawah ini :

  1. Tab Rekalkulasi Stock
    1. Tab ini langsung menjalankan proses rekalkulasi stock barang tertentu saja.

      2. Pilih gudang dan isikan periode, kode barang kemudian klik tombol Proses untuk melakukan rekalkulasi stock.

    2. Jika proses rekalkulasi stock telah selesai maka akan muncul pesan berikut :

      3. Klik tombol OK.

    3. Berikut hasil dari rekalkulasi stock :

  2. Tab Rekalkulasi Stock Tidak Normal
    1. Tab ini akan menampilkan list data barang dengan stock tidak normal kemudian dilakukan rekalkulasi stock per barang.

      2. Pilih Gudang dan periode kemudian klik tombol Proses untuk menampilkan list data.

    2. Berikut list data barang dengan stock tidak normal :

      3. Klik tombol Rekalkulasi untuk melakukan proses rekalkulasi stock.

    3. Berikut tampilan data yang sudah selesai dilakukan proses rekalkulasi stock :
