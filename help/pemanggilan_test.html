PEMANGGILAN TEST

Modul : SDM
Tentang : Pemanggilan Test
Isi :

Langkah-langkah melakukan Pemanggilan Test :

  1. Masuk ke menu, isikan departemen, nama lowongan, periode test dan umur seperti di bawah ini :

  2. Berikut Tampilan List Data Pelamar :

  3. Klik icon untuk cetak CV calon karyawan.
  4. Tick pada kolom Aksi untuk melakukan pemanggilan test pada calon karyawan yang dimaksud.
  5. Klik tombol Tolak untuk menolak pemanggilan test calon karyawan. Karyawan yang sudah ditolak tidak akan muncul lagi pada list data pemanggilan test.
