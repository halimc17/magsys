TRANSAKSI PACKING LIST

Modul : Pengadaan
Tentang : Transaksi Packing List
Isi :

Langkah-langkah input Transaksi Packing List :

Masuk ke menu, akan terdapat 2 Tab yaitu Tab Form dan Tab List Data seperti di bawah ini :

  1. Tab Form
    1. Lengkapi data Form sesuai dengan data yang dibutuhkan, No. Transaksi akan terbentuk secara otomatis :

    2. Akan muncul Form rincian untuk mencari No. PO yang akan melengkapi data tersebut, klik icon kaca pembesar berwarna biru seperti dibawah ini kemudian klik 'Cari No. PO' atau 'Cari kode barang' isi No.PO untuk melakukan pencarian, perhatikan Note yang menjelaskan mengenai pemilihan No.PO :

    3. Terdapat juga aplikasi untuk rincian berdasarkan Kode Barang dengan mengisi form seperti dibawah ini ketika klik icon kaca pembesar berwarna biru 'Cari Kode Barang' jika sudah mengisi klik tombol simpan dan selesai jika sudah menyelesaikan penginputan form

    4. Berikut tampilan rincian yang datanya berasal dari No.PO dan No.Kode Barang klik tombol simpan dan selesai jika sudah selesai mengubah atau memastikan datanya benar

  2. Tab List Data
    1. Klik List Data, untuk melihat data yang telah diinput sebelumnya, terdapat aplikasi untuk mencari data berdasarkan Kode PT , Periode dan No.Transaksi dengan memilih option seperti dibawah ini maka data akan otomatis keluar :

    2. Klik icon untuk edit Packing List data.
    3. Klik icon untuk menghapus Packing List Data
    4. Klik icon untuk memposting Packing List Data
    5. Icon belum posting merah akan berubah menjadi hijau ketika sudah melakukan posting
    6. Klik tombol PDF untuk melihat data Packing List dalam format PDF
