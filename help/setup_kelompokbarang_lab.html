SETUP KELOMPOK BARANG

Modul : Pengadaan
Tentang : Setup Kelompok Barang
Isi :

Langkah-langkah Setup Kelompok Barang :

  1. Masuk ke menu, lengkapi data yang dibutuhkan untuk Nomor Akun dapat memilih dengan klik tombol panah dan scroll down/up atau dapat mengetik cepat No. Akun maka data No. Akun yang dicari akan diarahkan seperti gambar dibawah ini, kemudian klik tombol Simpan untuk menyimpan data atau klik tombol Batal untuk membatalkannya.

  2. Setelah klik tombol Simpan maka akan muncul konfirmasi berikut :

    3. Klik tombol OK untuk menyimpan atau klik tombol Cancel untuk membatalkan.

  3. Kelompok Barang yang sudah diinput akan muncul pada List Data berikut :

  4. Klik icon untuk edit data kelompok barang.
  5. Klik icon untuk hapus data kelompok barang.
