SETUP KELAS POHON

Modul : Kebun
Tentang : Setup Kelas Pohon
Isi :

Langkah-langkah Setup Kelas Pohon :

  1. Masuk ke menu, akan terdapat form inputan untuk setting Setup Kelas Pohon seperti dibawah ini :

  2. Lakukan pengisian data kemudian klik tombol simpan maka data yang tersimpan akan muncul pada List Data dibawah ini :

    3. Klik icon edit untuk melakukan edit data Setup Kelas Pohon sebagai catatan untuk edit data hanya dapat mengedit nama seddangkan untuk Kelas Pohon tidak dapat diedit karena dikunci by system .

    Klik icon delete untuk meghapus data Setup Kelas Pohon .
