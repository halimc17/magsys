PROSES POSTING RKB

Modul : Kebun
Tentang : Proses Posting Rencana Kerja Bulanan (RKB)
Isi :

Langkah-langkah Proses Posting RKB :

  1. Input RKB (Rencana Kerja Bulanan) dilakukan melalui menu: Kebun–Proses–Posting RKB.

  2. Untuk memulai proses posting, pilih Periode (periode kerja) dan Kebun. Afdeling dan Kegiatan dapat secara optional dipilih untuk mengkhususkan pilihan. Klik Preview untuk menampilkan data. Icon Hijau menandakan bahwa data sudah diposting dan tidak dapat diganti lagi, icon Merah selain menandakan bahwa data belum diposting juga berguna untuk memposting data RKB secara satuan. Untuk memposting seluruh data RKB yang ditampilkan, klik Posting All.

    Bila tidak ada data yang ditampilkan, maka tidak ada data RKB yang dapat diposting untuk user yang bersangkutan (pilihan Manager pada inputan header RKB).
