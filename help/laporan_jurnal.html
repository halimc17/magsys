LAPORAN JURNAL

Modul : Keuangan
Tentang : Laporan Jurnal
Isi :

Langkah-langkah melihat Laporan Jurnal :

  1. Masuk ke menu, pilih perusahaan, pilih Perusahaan dan lokasi Perusahaan kemudian unit dan pilih periode laporan jurnal yang akan dilihat. Berikut tampilannya :

    2. Pilihan Rev untuk jurnal yang ada revisi, jika tidak ada revisi pilih Rev = 0. Klik tombol proses untuk melihat laporan jurnal.

  2. Berikut tampilan laporan jurnal :

  3. Klik tombol Excel atau klik icon untuk mellihat laporan dalam format Excel.
  4. Klik tombol PDF atau klik icon untuk mellihat laporan dalam format PDF.
