Modul : SDM
Tentang : Perubahan Format Slip Gaji Non Staff
Isi :

Berikut hal-hal yang harus dilakukan sebelum melakukan proses gaji:

  1. Menonaktifkan semua angsuran
    1. Menu : SDM > Transaksi > Angsuran
    2. Dilakukan oleh KASIE
    3. Ubah status angsuran menjadi non active :

  2. Setup Gaji Pokok
    1. Menu : SDM > Setup > Gaji Pokok
    2. Dilakukan oleh KASIE
    3. Pada setup gaji pokok hanya cukup input komponen gaji pokok saja :

      4. Tidak perlu memasukkan komponen SP/SW Kopkar Minanga,PHBI,Tunjangan Masa Kerja. Cukup memasukkan komponen gaji pokok saja.

  3. Input Potongan
    1. Menu : SDM > Transaksi > Administrasi Personalia > Potongan
    2. Dilakukan oleh KASIE
    3. Lengkapi form header dan detail berikut :

  4. Lakukan proses gaji.
