TRANSAKSI JURNAL ADJUSTMENT AUDIT

Modul : Keuangan
Tentang : Transaksi Jurnal Adjustment Audit
Isi :

Langkah-langkah Transaksi Jurnal Adjustment Audit :

  1. Masuk ke menu maka akan muncul daftar header dan form header seperti di bawah ini :

  2. Klik icon untuk menambahkan header baru. Berikut tampilan form header :

    3. Lengkapi form seperti di atas kemudian simpan. No Jurnal Adjustment Audit akan terbentuk secara otomatis oleh sistem.

  3. Setelah data header disimpan, akan muncul Form Jurnal Detail seperti di bawah ini :

    4. Lengkapi data form jurnal detail di atas kemudian simpan.

  4. Data jurnal detail yang sudah disimpan akan muncul pada tabel jurnal detail di bawah ini :

  5. Lakukan kembali langkah no. 3 untuk membuat jurnal kredit pada form jurnal detail sehingga pada Tabel Jurnal Detail akan terdapat jurnal debet dan kredit seperti di bawah ini :

  6. Untuk memastikan jurnal balance, lihat pada daftar header untuk no. jurnal yang dimaksud kemudian arahkan kursor pada baris jurnal yang dimaksud pada kolom Debet atau Kredit seperti di bawah ini :

    7. Jika sudah balance maka selisih 0, jika belum balance maka baris jurnal tersebut akan berwarna merah dan saat kursor diarahkan akan muncul jumlah selisihnya seperti di bawah ini :

  7. Lakukan langkah berikut untuk edit data header jurnal :
    1. Klik pada baris jurnal yang akan diedit seperti di bawah ini :

    2. Setelah klik jurnal yang dimaksud maka data header tersebut akan muncul pada form header di bawah ini :

      3. Klik tombol Simpan setelah selesai edit data header.

  8. Lakukan langkah berikut untuk edit detail jurnal :
    1. Klik pada baris jurnal yang akan diedit seperti di bawah ini :

    2. Setelah klik jurnal yang dimaksud maka data detail jurnal tersebut akan muncul pada Tabel Jurnal Detail seperti di bawah ini :

    3. Klik icon di atas untuk edit detail jurnal :

      4. Klik tombol Simpan setelah selesai edit detail jurnal.

    4. Klik icon untuk hapus detail jurnal.
  9. Lakukan langkah berikut untuk hapus header jurnal :
    1. Hapus terlebih dahulu detail jurnal kemudian hapus header jurnal dengan klik icon pada daftar header.

      2. Jika detail jurnal belum dihapus maka header jurnal tidak dapat dihapus.

  10. Klik icon untuk cetak jurnal dalam format PDF.
