Langkah-langkah melihat Laporan Summary Progress SPK :

Masuk ke menu, akan muncul 2 tab yaitu tab Summary Progress SPK dan tab Summary Progress SPK per Tahun seperti di bawah ini :

  1. Summary Progress SPK
    1. Pilih periode, regional dan unit kemudian klik tombol Preview untuk menampilkan Laporan.

      2. Laporan ini dapat ditampilkan dalam format Excel dan PDF dengan klik tombol Excel atau tombol PDF di atas.

  2. Summary Progress SPK per Tahun
    1. Pilih tahun dan kontraktor kemudian klik tombol Preview untuk menampilkan Laporan.

      2. Laporan ini dapat ditampilkan dalam format Excel dengan klik tombol Excel di atas.
